[Source: Jordan Career Summary | Section: Existing system improvements > Signage system modernization (Dick’s Sporting Goods)]

The POC was quickly approved, and I began turning it into a production‑grade system. I developed the WCF service that became the backbone of the platform, generating tens of thousands of PDF files per day using load‑balanced servers in production. This service integrated data from a SQL Server signage database with the appropriate PDF templates stored on a shared drive, producing pricing and promotional signage across three major brands.

Alongside the new service and GUI, both corporate and store users received upgraded ASP.NET intranet applications for managing signs. These updates improved usability and significantly boosted performance through optimized SQL procedures and queries used for CRUD operations on SQL Server and read operations on Oracle databases. The performance improvements also benefited the Windows CE mobile devices used in stores to scan and print item pricing signs, as well as the newer Android devices that received an updated mobile signage application.
